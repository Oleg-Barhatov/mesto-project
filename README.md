# Mesto

## Описание проекта

 В проекте место закрепили навыки адаптивной верстки и поработали с формами ввода данных пользователем. Функциональность проекта будет реализована при помощи JavaScript.

## Используемые технологии

1. HTML5
2. CSS3
3. FlexBox
4. Grid
5. FIgma
